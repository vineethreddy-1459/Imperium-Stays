# 🌿 Imperium Stays – Nature Styled Luxury Hotel Website

Welcome to **Imperium Stays**, a beautifully designed **nature-inspired luxury hotel website project**. This project showcases a modern hotel booking interface built using **HTML, CSS, and JavaScript**, focusing on elegance, simplicity, and user experience.

---

---

## 🏨 About Imperium Stays

**Imperium Stays** is a fictional luxury hotel designed with a **nature + premium hospitality theme**. The website represents a real-world hotel booking system with multiple pages and interactive UI.

This project is created for:
- 🎓 Academic / College project submission
- 💻 Frontend development practice
- 🎨 UI/UX design showcase
- 🧑‍💻 Portfolio project

---

## ✨ Theme Concept

🌿 Nature-inspired luxury hotel  
🏨 Modern hospitality experience  
🌸 Calm, elegant, and premium design  
🌎 Clean UI with responsive layout

## 📄 Website Pages

This project contains multiple structured pages:

### 🏠 Home Page (`index.html`)
- Hero section with welcome message
- About preview
- Featured rooms
- Hotel facilities
- Guest testimonials
- Navigation bar + footer

---

### 📖 About Page (`about.html`)
- Hotel introduction
- Mission & Vision
- Company values
- Why choose Imperium Stays

---

### 🛏️ Rooms Page (`rooms.html`)
- Deluxe Room – ₹2,500/night
- Executive Room – ₹4,000/night
- Family Suite – ₹6,000/night
- Presidential Suite – ₹8,500/night
- Booking buttons for each room

---

### ✨ Services Page
- Showcases the premium services offered by Imperium Stays.
- Highlights the luxury restaurant and fine dining experience.
- Features the swimming pool and spa facilities.
- Displays the modern fitness center.
- Includes conference and event hall information.
- Offers free high-speed Wi-Fi.
- Highlights 24/7 room service for guests.
- Designed to provide a clear overview of hotel amenities.

### 🖼️ Gallery Page (`gallery.html`)
- Hotel lobby
- Rooms showcase
- Restaurant
- Swimming pool
- Gym & spa
- Conference hall
- Garden & exterior

---

### 🏨 Booking Page
- Allows guests to book their stay at Imperium Stays.
- Collects guest details (Name, Email & Phone Number).
- Lets users choose their preferred room type.
- Includes Check-In and Check-Out date selection.
- Allows guests to specify the number of guests.
- Provides a section for special requests.
- Simple, responsive, and user-friendly booking form.

---

### 📞 Contact Page (`contact.html`)
- Booking form with validation
- Name, email, phone inputs
- Check-in / check-out dates
- Room selection dropdown
- Contact details section
